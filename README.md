# pseudocodigo
Pseudocodigo LaTeX package


## Exemplo 1

```LaTeX
\begin{algorithmic}
\Algoritmo[nome]{}
\Se [ $x > 0$ ]
  \Imprimir \str{teste}
\FimSe
\FimAlgoritmo
\end{algorithmic}
```
